# University Refactoring Laboratory

This package contains:
- A legacy Java starter project
- A complete laboratory manual for students
- Files intended for use in IntelliJ IDEA

## Quick start
1. Extract the ZIP file.
2. Open IntelliJ IDEA.
3. Choose **Open** and select the `UniversityRefactoringLab` folder.
4. Open `src/Main.java`.
5. Run `Main.java`.
6. Follow the laboratory script in `Refactoring_Lab_Manual.docx`.

## Folder layout
- `src/` -> starter Java files
- `Refactoring_Lab_Manual.docx` -> detailed lab script and procedure
- `README.md` -> this file
